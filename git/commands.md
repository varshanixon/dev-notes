# GIT COMMANDS

- `git init` : for initializing git repository in current working directory

- `git add` : move changes from working directory to staging area. `git add -A` moves all the changes made to staging area.

- `git commit` : save changes from staging area to local repository. `git commit -m "commit message"` is used to give commit message along with the command.

- `git status` : lists all the untracked files

- `git log` : shows all the commits made

- `git clone` : to get the exact copy of remote repository

- `git pull` : to fetch and download content from remote repository and immediately update local repository to match that content.It is a combination of `git fetch` (downloads content from remote repo) and `git merge` (merge the remote contents to local repo) 

- `git push origin main` : to move the contents in the local repo to remote repo

- `git remote add origin <https link of remote repo>` : to connect remote repo with local repo. Here 'origin' is shortform of URL.
